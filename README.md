# ✨ MapleStory - Goddess Tower Sealed Room Solver

A lightweight solver for the sealed room puzzle in the Tower of Goddess.

---

## ✨ Overview

A simple tool designed to quickly determine  
the correct player distribution across platforms **1 / 2 / 3**.

No more trial and error.  
No more second guessing.  
Just input, and get the answer ✨

## ✨ Features

- 🔢 Supports all valid sealed room combinations
- ⚡ Real-time calculation
- 🧩 Works with partial inputs
- 🎯 Automatically narrows down possible solutions
- 👆 Click or type input (0 / 1 / 2)
- 🪟 Picture-in-Picture (PiP) mode for quick access
- 📱 Mobile-friendly interface

---

## 🧠 How It Works

The tool simulates all possible platform combinations  
with a total of 4 players distributed across 3 platforms.

Based on the NPC responses from:

- A — No one stands  
- B — Leader stands on platform 1  
- C — Leader stands on platform 2  
- D — Leader stands on platform 3  

It filters out invalid cases and returns the remaining valid solutions.

---

## ⚠️ Notes

- NPC responses only range from **0 to 2**
- PiP mode requires Chromium-based browsers (Chrome / Edge)
- Firefox and Safari do not support PiP
